# Opencv Cplusplus Example

simple opencv c++ example

## 1. Install opencv

### 1.1. Install opencv on ubuntu

```bash
sudo apt-get install libopencv-dev
```

### 1.2. Install opencv on mac

```bash
brew install opencv
```

## 2. Compile

```bash
make
```

## 3. Run

```bash
./main
```
